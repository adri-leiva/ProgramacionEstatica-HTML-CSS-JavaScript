* Formato de texto

    <strike> tachar texto (stilo fisico) </strike>

    <strong> simil negrita,texto mucho mas importante (Estilo logico) </strong>
    <b> simil negrita,texto mucho mas importante (Estilo fisico) </b>

    <big> Aumentar el tamaño del texto (Estilo fisico) </big>
    <small> Reduce el tamaño del texto </small>

    <u> Texto subrayado (Estilo fisico) </u>

    <sub> Texto tipo subindice (Estilo fisico) </sub>
    <sup> Texto como superindice (Estilo fisico) </sup>

    <i> Cursiva - Marca el texto como importante (Estilo fisico) </i>
    <em> Cursiva - Marca el texto como importante (Estilo logico) </em>

    <var> Lugas donde escribe el usuario,variables,etc (Estilo logico)</var>
    <cite> introduce una cita/referencia (Estilo logico) </cite>
    <dfn> introducir una definición (Estilo logico) </dfn>

    <code> simil monoespaciado, Delimita el texo considerado un fragmento de código fuente (Estilo logico) </code>
    <tt> simil monoespaciado, Delimita el texo considerado un fragmento de código fuente (Estilo fisico) </tt>

ENLACES 
Atributos propios	
- name = "texto" - Permite nombrar al enlace para que se pueda acceder desde otros enlaces
- href = "url" - Indica la URL del recurso que se quiere enlazar
- hreflang = "codigo_idioma" - Idioma del recurso enlazado
- type = "tipo_de_contenido" - Permite "avisar" al navegador sobre el tipo de contenido que se enlaza (imágenes, archivos, etc.) para que pueda preparase en caso de que no entienda ese contenido
- rel = "tipo_de_relacion" - Describe la relación del documento actual con el recurso enlazado
- rev = "tipo_de_relacion" - Describe la relación del recurso enlazado con el documento actual
- charset = "tipo_de_charset" - Describe la codificación del recurso enlazado
